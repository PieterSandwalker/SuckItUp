Suck it Up!

Build A.1.0

Controls:
W : forward (except in Tron)
S : backward (except in Tron)
A,D : rotate
hold X : respawn

Instructions:
To play 'Suck it Up!', each player must be logged into a valid Steam account on a different computer. 

To start the game open the 'WindowsNoEditor' folder and run the executable: 'RoombaWars'

The Alpha Build version will take you through three maps, one of each game type and one of each location.
The combination of the two will be random, but you will be able to try each game mode.
